# PiggyCell 自动签到和游戏机器人

这是一个用于PiggyCell网站的自动签到和游戏机器人，支持多账户、定时签到、自动游戏和代理配置。

## 功能特点

- ✅ 多账户支持
- ✅ 定时自动签到
- ✅ 自动游戏功能
- ✅ 代理支持
- ✅ 详细日志记录
- ✅ 配置化管理
- ✅ 错误重试机制
- ✅ Token有效性检测

## 安装依赖

```bash
pip install -r requirements.txt
```

## 文件结构

```
piggycell/
├── config/
│   ├── config.json         # 主配置文件
│   └── accounts.json       # 账户信息
├── proxy.txt              # 代理配置
├── bot.py       # 主程序
├── requirements.txt       # 依赖包
└── README.md             # 使用说明
```
